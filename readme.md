# Simulation
Using [LTSpice](https://www.analog.com/en/resources/design-tools-and-calculators/ltspice-simulator.html) for circuit simulations. It's a free and very flexible software used for analog and digital circuits. Another feature LTSpice provides is the ability to add in other circuits to the framework and use any component's simulation information.
